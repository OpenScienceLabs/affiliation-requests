# OSL Affiliation Acknowledgement

Accepted projects should acknowledge OSL affiliation in their README and public
documentation where applicable. The acknowledgement should be accurate, modest,
and clear that the project remains independently governed and maintained.

## Suggested acknowledgement

```markdown
## Open Science Labs Affiliation

This project is affiliated with Open Science Labs (OSL). Affiliation means that
the project aligns with OSL's mission and is connected to the OSL ecosystem,
while governance, roadmap, maintenance, and releases remain the responsibility
of the project maintainers.
```

## Allowed phrases

These phrases are generally acceptable when true:

- OSL affiliated project
- Affiliated with Open Science Labs
- Connected to the Open Science Labs ecosystem

## Disallowed phrases without separate written permission

Do not use phrases that imply ownership, certification, guarantees, sponsorship,
or control by OSL, such as:

- OSL certified
- OSL approved software
- OSL maintained project
- Official OSL project
- OSL guarantees this project
- OSL funded project
- OSL-sponsored project

## Examples

Acceptable:

```markdown
Project Example is affiliated with Open Science Labs. The project is maintained
by the Project Example maintainers.
```

Not acceptable:

```markdown
Project Example is an official OSL project maintained and certified by Open
Science Labs.
```

## When affiliation changes

If affiliation is paused, removed, or changed, update or remove the
acknowledgement promptly. If the project is reactivated later, add accurate
wording again after OSL confirms reactivation.
